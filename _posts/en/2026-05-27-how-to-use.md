**PDF editing with UniFicarPDF is completely anonymous.** The process begins when a PDF file is uploaded for editing; once it has been processed and downloaded, it is immediately deleted. For more details, the “Privacy at a Glance” page provides a complete explanation of how this anonymous process works. [It's fast, simple, and secure!](/en/blog/is-it-safe-to-upload-files-to-unificarpdf/)

## Getting Started

Visit *UniFicarPDF.com*. On the homepage, you will find all available PDF tools listed. Simply choose the tool that best matches your needs:

- Merge PDF – combine multiple PDF documents into a single PDF file.
- Split PDF – separate pages from a PDF document or split large PDF files into smaller files.
- Compress PDF – reduce file size before sending documents by email.
- Convert PDF – convert PDF to Word, Word to PDF, PDF to JPG, or JPG to PDF.
- Reorder Pages – arrange PDF pages in any order you want.
- Delete Pages – remove pages you no longer want to keep in the document.
- Rotate PDF – rotate pages that were scanned upside down or sideways.

Select the tool you want to use. Once selected, the dedicated tool page will open, where you will see an empty upload area. Only the selected PDF tool will be available on that page.

## Using the Tools

Once you have opened the tool page, follow these steps:

Upload your PDF file by dragging and dropping it into the upload area, or click the upload button to select a file from your device. If you want to merge multiple PDF documents, you can drag and drop all files at the same time.

Adjust the PDF settings if needed. Depending on the selected tool, you may reorder pages, choose page ranges, compress files, or select the desired output format. In most cases, the default settings work perfectly.

Click the processing button and wait a few seconds while your PDF file is processed.

Download your completed PDF file.

> This process is similar for all tools available on the website.

## File Size and Format: What You Should Know

UniFicarPDF supports the standard PDF format as well as other popular file formats that can be converted into PDFs. If you are working with large PDF files, compression can significantly reduce their size. For example, a scanned 15 MB PDF may be reduced to approximately 3–4 MB.

Converted files preserve nearly all of their original formatting. In some cases, pages containing images may shift slightly, but major formatting issues are uncommon.

The website is available in multiple languages and is designed to be easy to navigate, allowing most users to find the tool they need within seconds.

## Works on Any Device

There is no need to install any software because UniFicarPDF runs entirely in your web browser. Whether you use a laptop, smartphone, or tablet, the experience remains fast, simple, and consistent across all devices.
